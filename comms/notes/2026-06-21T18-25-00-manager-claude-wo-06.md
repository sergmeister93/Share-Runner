## WO-06 — Preloader scene (queued behind WO-07)

**Goal.** A dedicated loading scene that loads every manifest-derived asset, reports progress, and
hands off to the Level scene. Pick this up after you submit WO-07.

**The asset adapter is ready (WO-05 merged).** Use it — do NOT read manifests or hardcode paths
yourself:
- `loadManifestBundle()` (`src/data/assetManifests.ts`) → `await` it in `init`/`preload`.
- `buildAssetCatalog(bundle)` + `queueLoads(this.load, catalog)` (`src/systems/AssetCatalog.ts`)
  to enqueue all images/spritesheets/audio.
- Stash the built `catalog` on the registry (`this.registry.set('assetCatalog', catalog)`) so the
  Level scene reads placements/keys from it.

**Do (lane: `src/scenes/**`, `src/ui/**`).**

  `preloader:start { levelId }` (LEVEL_ID from Constants).

- On `complete` emit `asset:load-complete { levelId }` then `this.scene.start(SCENE_LEVEL)`.
- Tear down in `shutdown()`.
